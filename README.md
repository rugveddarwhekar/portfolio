# rugved_s_darwhekar // Personal Portfolio

A terminal-inspired personal portfolio site built with vanilla HTML, CSS, and JavaScript. Designed to reflect a systems engineering mindset — minimal dependencies, maximum signal.

## Live

[rugvedd.com](https://rugvedd.com)

## Features

- **Terminal aesthetic** — monospace typography (JetBrains Mono), CRT scanline overlay, glitch effects on the hero name
- **Typewriter rotation** — cycles through role titles with a blinking cursor
- **Scroll-triggered reveals** — elements animate into view using `IntersectionObserver`
- **Animated skill bars** — progress bars fill on scroll for the toolkit section
- **Active nav tracking** — sidebar and topbar highlight the current section on scroll
- **Live clock** — real-time 24h clock in the topbar
- **Fully responsive** — sidebar collapses on mobile, grid layouts adapt
- **Zero dependencies** — no frameworks, no build tools, single `index.html`

## Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | Vanilla CSS (custom properties, grid, clamp) |
| Logic | Vanilla JavaScript |
| Font | [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts |
